[python]输出一个简单的进度条

周一 13 一月 2014

使用 sys.stdout 或 sys.stderr 就可以输出一个简单的进度条：

import time
import sys

f = sys.stdout

for n in range(1, 101):
    s = ('#' * n).ljust(100, '-')
    f.write(s)
    f.flush()
    time.sleep(0.3)
    f.write('\r')
f.write('\n')

[python]指定 Socket connect 方法的超时时间

周一 18 十一月 2013

通过设置 socket.settimeout 来实现超时功能：

import select
import socket

HOST = '127.0.0.1'
PORT = 8000
timeout = 60 * 1   # 1 分钟

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
# 设置连接超时
s.settimeout(10)
s.connect((HOST, PORT))
# 恢复默认超时设置
s.settimeout(None)
s.connect((HOST, PORT))
s.sendall('msg')
