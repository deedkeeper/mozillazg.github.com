[linux]创建指定大小的文件

周六 03 八月 2013
By mozillazg

In linux.

tags: linux

比如创建 10M 大小的文件：

# 创建大小为 10M 的文件：abc.exe
dd if=/dev/zero of=abc.exe bs=1M count=10

# 只生成属性为 10M 大小的文件，但不占用实际硬盘空间
dd if=/dev/zero of=abc.exe bs=1M seek=10 count=0

  2. [linux]screen 常用命令

    周四 16 五月 2013
    By mozillazg

    In linux.

    tags: linuxscreen

    通过 screen 命令可以让终端命令在我们断开远程服务器后依旧运行而不会中断。

    安装 screen 命令

    redhat/centos:

    yum install screen

    ubuntu:

    sudo apt-get install screen

    常用的 screen 命令

    新建一个名为 django 的 screen shell：

    screen -S django

    新建一个名为 django 的后台 screen shell：

    screen -dS django

    查看所有的 screen shell:

    $ screen -ls
