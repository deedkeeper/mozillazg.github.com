微信开发过程中遇到的问题

Date 周一 08 六月 2015 Category others Tags 微信

JSSDK

invalid signature, 排查:

前端：

  • url 参数动态获取

  • 注意参数大小写:

    appId: '',         //  I 大写
timestamp: 134xxx, // 全小写, 必须与后端签名时 timestamp 的值一致
nonceStr: '',      //  ** 尤其注意 S 是大写 **, 必须与后端签名时 noncestr 的值一致

后端：

  • 确保 jsapi_ticket 未过期

  • 确认参与签名的参数，注意大小写:

    noncestr        // 全小写，** 尤其注意 s 是小写 **
jsapi_ticket    // 全小写
timestamp       // 全小写
url             // 小写，必须是前端动态获取

  • 签名算法 * 是否按字段名的ASCII 码从小到大排序 * 使用 sha1 签名

修复无法启动 Oracle SQL Developer，提示：could not install some modules

Date 周日 02 十一月 2014 Category others

今天启动 Oracle SQL Developer时，提示如下错误：

Warning - could not install some modules:

导致不能成功启动 Oracle SQL Developer。

最后的解决办法是（ 注意，这个办法会导致丢失所有之前定义的数据库连接配置 ）：

删除 c:\Users\yourusername\AppData\Roaming\SQL Developer\ 目录下的 system4.0.0.12.84 文件夹 。

修复启动 sqldeveloper 时出现“Unable to create an instance of the Java Virtual Machine Located at path”错误

Date 周四 17 七月 2014 Category others Tags sqldeveloper

刚才打开 sqldeveloper 时提示如下错误：

Unable to create an instance of the Java Virtual Machine
Located at path:
x:\xxxx\jre\bin\client\jvm.dll

最后的解决办法是：编辑 sqldeveloper.exe 所在目录下的 ide\bin\ide.conf 文件， 将 38 行左右的 AddVMOption -Xmx800M 改为 AddVMOption -Xmx512M

# AddVMOption  -Xmx800M
AddVMOption -Xmx512M
