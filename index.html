前端：

  • url 参数动态获取

  • 注意参数大小写:

    appId: '',         //  I 大写
timestamp: 134xxx, // 全小写, 必须与后端签名时 timestamp 的值一致
nonceStr: '',      //  ** 尤其注意 S 是大写 **, 必须与后端签名时 noncestr 的值一致

后端：

  • 确保 jsapi_ticket 未过期

  • 确认参与签名的参数，注意大小写:

    noncestr        // 全小写，** 尤其注意 s 是小写 **
jsapi_ticket    // 全小写
timestamp       // 全小写
url             // 小写，必须是前端动态获取
