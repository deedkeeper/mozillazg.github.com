[django]修复 MemcachedKeyCharacterError

Date Tue 09 July 2013 Category django Tags django / memcached

错误详情

MemcachedKeyCharacterError at /

Control characters not allowed

出现这个错误是因为 key 值不合法：长度大于 250、包含空格或控制符

解决

去掉非法字符或将 key 进行编码(base64/md5/sha1/...)

[python]设置 gunicorn 绑定多个地址

Date Mon 08 July 2013 Category python Tags python / gunicorn

有两种方式可以设置 gunicorn 绑定多个地址。

命令行参数

gunicorn -b unix:/tmp/gunicorn.sock -b 127.0.0.1:8000 wsgi:app

gunicorn --bind unix:/tmp/gunicorn.sock --bind 127.0.0.1:8000 wsgi:app

配置文件

bind = 'unix:/tmp/gunicorn.sock','127.0.0.1:8000','0.0.0.0:1234'

[python] 发送邮件时设置邮件的优先级/重要性

Date Tue 02 July 2013 Category python Tags python / email / X-Priority

设置邮件 header 中 X-Priority 的值就可以了：

X-Priority 的值 含义
"1" 最高级别（重要性高）
"2" 介于中间
"3" 普通级别（不提示重要性）
"4" 介于中间
"5" 最低级别（重要性低）

Cygwin 中的软件包管理工具： apt-cyg

Date Sun 23 June 2013 Category others Tags Cygwin

apt-cyg 可以在 Cygwin 中实现类似 linux 下的 yum 或 apt-get 的功能。

主页

http://code.google.com/p/apt-cyg/

安装

$ svn --force export http://apt-cyg.googlecode.com/svn/trunk/ /bin/
$ chmod +x /bin/apt-cyg

使用

  • apt-cyg install <package names> # 安装软件包

    apt-cyg install bind-utils

  • apt-cyg remove <package names> # 移除软件包

  • apt-cyg update # 更新 ...
