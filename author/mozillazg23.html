周二 20 五月 2014

  • 修改 vsftpd.conf，启用被动模式，指定端口:

    pasv_min_port=12000
pasv_max_port=12199
pasv_enable=YES

  • 配置防火墙 iptables，允许端口访问:

    iptables -I INPUT -p tcp --dport 12000:12199 -j ACCEPT

[python]输出一个简单的进度条

周一 13 一月 2014

使用 sys.stdout 或 sys.stderr 就可以输出一个简单的进度条：

import time
import sys

f = sys.stdout

for n in range(1, 101):
    s = ('#' * n).ljust(100, '-')
    f.write(s)
    f.flush()
    time.sleep(0.3)
    f.write('\r')
f.write('\n')
