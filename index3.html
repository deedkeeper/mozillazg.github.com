周三 06 一月 2016

函数注解（Function Annotations）

函数注解语法 可以让你在定义函数的时候对参数和返回值添加注解：

def foobar(a: int, b: "it's b", c: str = 5) -> tuple:
    return a, b, c
  • a: int 这种是注解参数
  • c: str = 5 是注解有默认值的参数
  • -> tuple 是注解返回值。

注解的内容既可以是个类型也可以是个字符串，甚至表达式：

def foobar(a: 1+1) -> 2 * 2:
    return a
