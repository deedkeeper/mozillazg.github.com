比如，下面的模板代码:

<script id="template" type="x-tmpl-mustache">
    {{#stooges}}
      <li>{{name}}</li>
    {{/stooges}}
    {% test %}
</script>

Date Sat 28 February 2015 Category go

安装

go get github.com/laher/goxc

使用

所有命令皆在 main 包所在目录下执行

基本使用

goxc -d=build -pv=0.1.0

上面命令的意思是：将编译好的文件保存到 ./build 目录下，指定文件名包含的版本号为 0.1.0
