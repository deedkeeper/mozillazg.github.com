    本文将讲述如何使用 South 重命名表的字段名。

    假设应用 foobar 有个 model:

    class Foo(models.Model):
    foo = models.IntegerField()

    想要将 foo 字段名改为 bar。

    1. 改好 models.py

      class Foo(models.Model):
    bar = models.IntegerField()

    2. 执行 south 命令

      # python manage.py schemamigration yourapp rename_field_foo_to_bar --empty
