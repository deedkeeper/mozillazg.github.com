[python]设置 gunicorn 绑定多个地址

周一 08 七月 2013

有两种方式可以设置 gunicorn 绑定多个地址。

命令行参数

gunicorn -b unix:/tmp/gunicorn.sock -b 127.0.0.1:8000 wsgi:app

gunicorn --bind unix:/tmp/gunicorn.sock --bind 127.0.0.1:8000 wsgi:app

配置文件

bind = 'unix:/tmp/gunicorn.sock','127.0.0.1:8000','0.0.0.0:1234'

周二 02 七月 2013

设置邮件 header 中 X-Priority 的值就可以了：

X-Priority 的值 含义
"1" 最高级别（重要性高）
"2" 介于中间
"3" 普通级别（不提示重要性）
"4" 介于中间
"5" 最低级别（重要性低）

[python]解决编译安装 lxml 时提示：libxml/xmlversion.h:No such file or directory

周日 23 六月 2013

编译安装 lxml 时，出现了如下错误：

libxml/xmlversion.h: No such file or directory

出现这个错误是因为有些依赖包没有安装：

  • libxml2
  • libxml2-dev
  • libxslt
  • libxslt-dev

只要安装好要上面的依赖包就可以了
