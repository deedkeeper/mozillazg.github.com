[linux] CentOS 下安装 percona-toolkit

周一 24 三月 2014

percona-toolkit 是一组高级命令行工具的集合， 用来执行各种复杂和麻烦的 MySQL 操作。

下面记录一下在 CentOS 下如何通过 rpm 安装 percona-toolkit:

  1. 安装依赖
    yum install perl perl-IO-Socket-SSL perl-Time-HiRes

  2. 下载 rpm 包
    wget percona.com/get/percona-toolkit.rpm

  3. 安装 rpm 包
    rpm -ivh percona-toolkit-*.rpm

[MySQL]修复 DatabaseError: (1061, "Duplicate key name 'bar_xxx_uniq'") 错误

周二 10 九月 2013

当 MySQL 数据库的索引 key 出现重复时，就会提示类似的错误：

DatabaseError: (1061, "Duplicate key name 'abc_xxx_uniq'")

解决办法是移除重复的索引 key:

ALTER TABLE `foo`.`bar` DROP INDEX `bar_xxx_uniq`;
