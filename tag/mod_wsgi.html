[django]使用 apache + mod_wsgi 部署 django

Date Thu 31 January 2013 Category django Tags python / django / apache / mod_wsgi

本文测试环境：linux mint，python 2.7，django 1.4

安装依赖

$ sudo apt-get install apache2
$ sudo apt-get install libapache2-mod-wsgi

设置 django

本例中项目名为 bbs:

$ pwd
/var/www/bbs

$ tree . -d
.
|-- bb  # 应用
|-- bbs  # settings.py wsgi.py
|-- static  # 静态文件
|   `-- css
`-- templates  # TEMPLATE_DIRS

配置 settings：

TEMPLATE_DIRS = (
    '/var/www/bbs/templates',  # 这里要是绝对路径
)
