  2. [linux]screen 常用命令

    周四 16 五月 2013
    By mozillazg

    In linux.

    tags: linuxscreen

    通过 screen 命令可以让终端命令在我们断开远程服务器后依旧运行而不会中断。

    安装 screen 命令

    redhat/centos:

    yum install screen

    ubuntu:

    sudo apt-get install screen

    常用的 screen 命令

    新建一个名为 django 的 screen shell：

    screen -S django

    新建一个名为 django 的后台 screen shell：

    screen -dS django

    查看所有的 screen shell:

    $ screen -ls

  3. [django]__unicode__: Bad Unicode data 问题

    周日 05 五月 2013
    By mozillazg

    In django.

    tags: djangopython

    问题：

    >> models.Foo.objects.all()
>> [<Foo: [Bad Unicode data]>]

    解决办法：

    更改 model 的 __unicode__ 方法：

    # Before
def __unicode__(self):
    return '%s' % self.get_foo_display()  # it's bad

# To
def __unicode__(self):
    return u'%s' % self.get_foo_display()  # it's Ok
